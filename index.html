Ferramenta de controle de versão

**Ferramenta de controle de versão pode ser definido como um software que tem a finalidade de gerenciar diferentes versões no desenvolvimento de um documento qualquer, o GIT eh um exemplo desse tipo de ferramenta.

  • Git é um sistema de controle de versão distribuído e um sistema de gerenciamento de código fonte, com ênfase em velocidade.
